Next-Gen Architecture

⚡

Pure C++ Backend

We bypassed the Flutter UI thread entirely. The Libretro core runs in a dedicated NDK thread, rendering directly to memory buffers without Dart FFI overhead, completely eliminating Isolate boundary violations and SIGABRTs.

📡

External Display Projection

Project gameplay directly to a compatible television or monitor with native Android and iOS display support.

📺

Miracast Ready

Cast your gameplay to a Smart TV using Android's Presentation API. The C++ EGL renderer draws directly to the external display's SurfaceView, perfectly scaled and synced, while AAudio routes sound seamlessly.

🎮

Neon Gamepad

Your phone transforms into a sleek, neon-styled controller. Inputs are captured in the Flutter UI and instantly transmitted to the background C++ emulator loop for zero-latency responses.
